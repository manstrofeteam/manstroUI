表单

因公司项目中所有的表单都是水平排列设计的，故默认表单样式为水平排列。


<form class="ms-form">
    <div class="form-item">
        <label for="username">姓名：</label>
        <input type="text" name="username" id="username" class="form-control" placeholder="请输入姓名……">
    </div>
    <div class="form-item">
        <label for="pwd">密码：</label>
        <input type="password" name="pwd" id="pwd" class="form-control" placeholder="请输入密码……">
    </div>
    <div class="form-item">
        <label for="remarks">备注：</label>
        <textarea name="name" id="remarks" class="ms-textarea" placeholder="请输入备注……"></textarea>
    </div>
    <div class="form-item">
        <label>下拉列表：</label>
        <select class="form-control">
            <option>1</option>
            <option>2</option>
            <option>3</option>
            <option>4</option>
            <option>5</option>
        </select>
    </div>
    <div class="checkbox">
        <input id="agree" type="checkbox">
        <label for="agree">阅读协议并同意《**网站协议》</label>
    </div>
    <div class="radio">
        <input type="radio" name="myradio" id="radio1">
        <label for="radio1"> 单选一</label>
        <input type="radio" name="myradio" id="radio2">
        <label for="radio2">单选二</label>
    </div>
    <div class="form-item">
        <button class="ms-btn btn-default" type="reset">重置</button>
        <button class="ms-btn btn-primary" type="button">保存</button>
    </div>
</form>

按钮

可作为按钮使用的元素

最好使用 button 元素来获得在各个浏览器上获得相匹配的绘制效果

Link 

<a class="ms-btn btn-default" href="#" role="button">Link</a>
<button class="ms-btn btn-default" type="submit">Button</button>
<input class="ms-btn btn-default" type="button" value="Input">
<input class="ms-btn btn-default" type="submit" value="Submit">

带有预订样式的按钮

当然了如果按钮样式不符合实际开发效果，可做相应修改，依具体情况定。


<button type="button" class="ms-btn btn-default">（默认样式）Default</button>
<button type="button" class="ms-btn btn-primary">（首选项）Primary</button>
<button type="button" class="ms-btn btn-success">（成功）Success</button>
<button type="button" class="ms-btn btn-info">（一般信息）Info</button>
<button type="button" class="ms-btn btn-warning">（警告）Warning</button>
<button type="button" class="ms-btn btn-danger">（危险）Danger</button>
<button type="button" class="ms-btn btn-transparent">（透明）Transparent</button>

图片

响应式图片

通过为图片添加 .ms-img-responsive 类可以让图片支持响应式布局。其实质是为图片设置了 max-width: 100%;、 height: auto;属性，从而让图片在其父元素中更好的缩放。

<img class="ms-img-responsive" src="../ui/img/public/demo.jpg">

图片形状

通过为 <img> 元素添加不同的class类，可以让图片呈现不同的形状。

请时刻牢记：Internet Explorer 8 不支持 CSS3 中的圆角属性。

  • .am-radius 圆角
  • .am-round 椭圆
  • .am-circle 圆形，一般用于正方形的图片(你要觉得椭圆好看，用在长方形上也可以)
<img class="ms-radius" alt="140*140" src="../ui/img/public/demo.jpg" width="140" height="140">
<img class="ms-round" alt="140*140" src="../ui/img/public/demo.jpg" width="200" height="120">
<img class="ms-circle" alt="140*140" src="../ui/img/public/demo.jpg" width="140" height="140">
